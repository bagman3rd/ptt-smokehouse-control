# Live POS Integration Roadmap

## Build 8.0.2 — Foundation + Square
Shared connector contracts, tenant-scoped encrypted credentials, raw records, normalized sales, item mapping, manual/daily sync, webhook intake, sync history, and the first enabled Square connector.

## Build 8.1.0 — Square production hardening
Refresh-token rotation, payment/refund reconciliation, location selection hardening, 90-day backfill jobs, cursor checkpointing, retry/backoff, official report reconciliation, and Square sandbox certification evidence.

## Build 8.2.0 — Toast
Toast partner authentication, restaurant selection, menus, orders/checks/payments, incremental sync, reconciliation and customer authorization workflow.

## Build 8.3.0 — Clover + Lightspeed
OAuth connectors, catalog/order adapters, webhook/reconciliation coverage and provider-specific sandbox suites.

## Build 8.4.0 — TouchBistro + SpotOn + Revel
Partner-gated connectors, credentials workflow and production pilots.

## Build 8.5.0+ — Oracle Simphony, NCR Aloha, PAR Brink
Enterprise certification, reseller/customer configuration, optional local bridge where cloud APIs are insufficient.

No provider is shown as enabled until its actual authorization, import and reconciliation path exists.
