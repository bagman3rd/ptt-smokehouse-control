# Build 5.9.3 — Smoker Workflow Fix

Build 5.9.3 tightens the smoker setup workflow.

## Changes

- Manufacturer-published whole-chicken capacity now loads 1:1 as chicken / double-breast planning capacity.
- Southern Pride SPK-500 loads 20 briskets, 60 pork butts, 60 rib racks, and 70 chicken / double-breast units.
- Southern Pride MLR-150 loads 8 briskets, 24 pork butts, 24 rib racks, and 32 chicken / double-breast units.
- Ole Hickory EL-ED/X loads 40 briskets, 80 pork butts, 105 rib racks, and 72 chicken / double-breast units.
- Smoker Name label changed to Smoker Brand.
- Location changed from free text to dropdown.
- Cook window changed from free text to dropdown.

- Overnight only
- Same-day only
- All day / flexible
- Backup / overflow only
- Not currently active
